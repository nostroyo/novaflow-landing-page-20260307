# RacePact MVP (No-Login, Guest-First)

Playable static MVP for a fast F1-inspired racing experience.

## What is included

- Instant **guest race** (no login or account required)
- 8-driver grid (**you + 7 AI opponents**)
- F1-inspired core characteristics:
  - pace
  - tire strategy/risk (soft/medium/hard/inter)
  - weather impact (dynamic wet laps)
  - pit timing penalties and safety-car variance
  - overtaking windows
  - consistency variance
- Local replay loop with best finish saved in LocalStorage

## Run locally

Open `index.html` in any modern browser.

## Architecture (MVP)

- Static `index.html`, `styles.css`, `game.js`
- No backend required
- No auth dependency

## Deployment

Published via **GitHub Pages** from the `main` branch root.
